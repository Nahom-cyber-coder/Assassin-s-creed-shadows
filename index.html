About the Game

Assassin's Creed: Shadows is an upcoming game set in a dark and mysterious world. The game follows a new protagonist, a young assassin who uncovers a web of conspiracies spanning centuries. Set in an atmospheric environment, Shadows introduces advanced stealth mechanics, challenging parkour sequences, and a gripping story of betrayal and revolution.

A glimpse of the dark world in Assassin's Creed: Shadows.

Main Features

  • Rich storytelling with deep lore about the Assassin Order and the Templars.
  • Open-world gameplay with vast environments to explore.
  • Advanced stealth and parkour mechanics for a more immersive experience.
  • Multiple playable characters with unique skills and abilities.
